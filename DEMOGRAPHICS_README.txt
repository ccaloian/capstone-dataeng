Column name, type and description of the `us-cities-demographics.csv` dataset.

City - (String) Name of the city.

State - (String) Name of the state.

Median Age - (Float) Median age in the population in city.

Male Population - (Integer) Number (count) of male population in city.

Female Population - (Integer) Number (count) of female population in city.

Total Population - (Integer) Total population count in city.

Number of Veterans - (Integer) Number of veterans in city.

Foreign-born - (Integer) Number of foreign born people.

Average Household Size - (Float) Averge household size.

State Code - (String) Two character state code.

Race - (String) Race description.

Count - (Integer) Total number of people of given race in the city.
